# Primitive Data Types

**1. String**

- `"text"`
- `'text'`

**2. Number**

- 118
- 3.14

**3. Boolean**

- true
- false

**4. Undefined**

---

#### Number & String

```js
const calories = 180;
console.log(calories);

const calories2 = "180";
console.log(calories2);

// plus
console.log(calories + 1); // 181  (number)
console.log(calories2 + 1); // 1801 (string)
```

#### Boolean

```js
const showForm = false;
console.log(showForm);
```
