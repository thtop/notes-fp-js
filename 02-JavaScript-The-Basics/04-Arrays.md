# Arrays

- Contain zero or more other data types

```js
const meals = [
  { description: "Breakfast", calries: 460 },
  { description: "Snack", calries: 220 }
];

const meal1 = meals[0];
console.log(meal1);
```

#### Programmatically

- Add
- Update
- Remove

#### JavaScript Primitive Types

- String
- Number
- Boolean
- undefined

#### Comples Types
```js
// object literals
 { description: "Snack", calries: 220 }

 // Dates
 new Date(2020, 0, 1);

 // Arrays
 [mael1, meal2]

```
